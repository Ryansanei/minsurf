Differential Geometry · Open Source

minsurf

Discrete minimal-surface toolkit — soap-film form-finding, TPMS generator for 3-D printing, and cotangent-Laplacian flow with exact-solution validation.

74 tests passing Python 3.10 – 3.12 MIT License No heavy deps

Install

pip install minsurf            # core solver
pip install "minsurf[tpms]"    # + Gyroid / TPMS generator
🫧
Soap-Film Solver
Pick a wire frame — saddle ring, wavy ring, two-ring catenoid, or flat disk. Watch the mesh flow to zero mean curvature in real time. See the Goldschmidt collapse past ratio 1.3255.
Interactive · runs in browser
🔬
TPMS Explorer
GPU-raymarched Triply Periodic Minimal Surfaces — Gyroid, Schwartz-P, Diamond, Neovius. Adjust unit cells and level offset in real time. These are the gold-standard infill for FDM 3-D printing.
WebGL raymarching · 60 fps
🌀
Associate Family
Drag the θ slider to isometrically deform the catenoid into the helicoid — every intermediate surface is also minimal. Visualises the Weierstrass–Enneper representation.
Catenoid ↔ Helicoid morph
🏗️
Tensile Form-Finding
Click the plan grid to place mast supports at any height. Press Solve — the membrane relaxes to its minimal-area shape. Export the result as OBJ for use in Rhino or Blender.
Architectural membranes · OBJ export
